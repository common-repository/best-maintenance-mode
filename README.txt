== Description ==

Maintenance mode is the best plugin for coming soon style messages. You can still work with your website while users get "coming soon" message. Simply enable it from your dashboard and you can safely work on developing your website. We have tested maintenance plugin with most popular WordPress themes and it works flawlessly, however, if you notice any issues with the theme you are using please use support forum to let us know. 

= Some of the Maintenance mode plugin features: =

* Easy to use yet advanced interface for admins
* Exclude specific pages
* Robots meta tag for search engines
* Use Google analytics or any other visitors tracking code
* Completely customizable texts
* Integrated social networks
* Custom CSS field for complete customization
* Change background image
* Add blur effect for background
* Change logo
* Retina ready support for logo
* Customizable page title
* SEO optimized
* Translation ready

== Installation ==

Installing "Best Maintenance mode" can be done either by searching for "Best Maintenance mode" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
